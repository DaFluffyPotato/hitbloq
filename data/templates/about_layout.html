About

Hitbloq is a competitive ranking service for Beat Saber. Our goal is to provide a leaderboard and ranking system that anybody can use to create their own ranked ladders with a map pool of their choice. At the moment, Hitbloq is effectively an extension to ScoreSaber since we run off of their data. Any scores on ScoreSaber can be imported into Hitbloq. A large part of Hitbloq is the Hitbloq Discord. The Discord server is a space for suggestions and discussion while also being the location the voting and ranking processes for the main map pool take place.



FAQ

Do I need to use a mod?

Hitbloq pulls data from ScoreSaber using the ScoreSaber API. It does not take data directly from mods. However, a mod can tell Hitbloq to request an update from ScoreSaber when you finish a map, so it may be convenient to use one.

I just beat a song and requested an update, but the score isn't there. What do I do?

A few things can happen here. First, it could be that you used a modifier. All scores with modifiers are blocked on Hitbloq (due to some technicalities with how ScoreSaber works). Second, if you're using a mod, the update request after you beat a song could exceed Hitbloq's update request rate limits. If this is the case, wait a couple minutes and request an update from the website. Third, it's possible the action queue is busy with a large user addition and user updates may be delayed for a bit. This should be fixed soon. If all else fails, ping DaFluffyPotato on the Hitbloq Discord server.

I added myself, but my old scores didn't show up. What do I do?

User additions can take a while due to ScoreSaber's rate limits (a workaround is being developed). If a couple hours have passed and you're still missing scores, please ping DaFluffyPotato in the Hitbloq Discord server.

How to I find my Hitbloq page?

The easiest way is to use the search bar to search your Steam/Oculus name after your account has been added. Alternatively (if your name doesn't contain any normal characters), you can look for your score on a leaderboard for a map you've played or you can look for youself on a map pool leaderboard.

How to I make a map pool?

You can make and run your own map pool in the Hitbloq Discord server if you're a $5 patron.

How do I get the score banner / profile customization?

All patrons can customize their score banner and profiles from the Hitbloq Discord server.

Hitbloq Mod

The Hitbloq mod will automatically add your account to Hitbloq upon first startup. (Automatic user additions don't work yet. Give it a couple days. Use the section below for now.) You can get it here.

Please note that it may take a while for your ScoreSaber scores to fully import!

Alternatively, you can use the section below if you don't plan on using the mod.

Manually Add User

Please only add yourself. There are extremely strict limits on website-based user additions. (The Discord is less strict since it's easier to track.) Since Hitbloq uses Score Saber data, you need to have a Score Saber account. Enter your Score Saber profile URL below.

Example: https://scoresaber.com/u/76561198098172449
